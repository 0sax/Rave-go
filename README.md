# Rave-go

Rave-go is a Go library for the [API] for (https://flutterwavedevelopers.readme.io/v2.0/reference) flutterwave's [Rave by Flutterwave](https://rave.flutterwave.com/) payment platform.
#### Payment types implemented:
* Card Payments
* Preauth 
* Bank Account Payments


## Installation
To install, run

``` go get github.com/anjolabassey/Rave-go/rave ```

## Import Package
After the installation, you can import the library like this:

``` go
import (
    ...
    "github.com/anjolabassey/Rave-go/rave"
)
```

## Initialization
Go to [rave](https://rave.flutterwave.com/) for live, (https://ravesandbox.flutterwave.com/) for test and sign up.
This would provide you with a public and private key.


## Before you dive in!
Rave-go was built with **version 2.0** of Rave's API in mind. If you need to check the documentation for any information, make sure you're looking at https://developer.flutterwave.com/v2.0/reference#introduction
